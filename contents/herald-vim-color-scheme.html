----- title: "Herald Vim Color Scheme" content-type: project subtitle: "An easy to read, portable and pretty Vim Color Scheme" home: /herald-vim-color-scheme/ summary: "A dark, easy-to-read color scheme for the Vim text editor. It can be used with 255, 16 or 8 colors." active: true version: 0.2.1 -----

herald is a dark color scheme for Vim which aims to be easy to read, eye-appealing, portable on multiple terminals, and suitable for source code highlighting in multiple languages.

You can get the latest version HERE (direct link).

Changelog

  • v0.2.1
    • Improved readability in search highlights
    • Delimiters are now rendered like operators
  • v0.2.0
    • Support for 8 and 16 color modes
    • Fixed completion menu colors
    • Improved readability for Visual mode
    • Specified highlighting for all the most common Vim syntax groups
  • v0.1.0
    • Initial Release
