----- title: New CakePHP Manual (with associations!) content-type: article timestamp: 1145342820 tags: cakephp -----

Gustavo Carreno just announced a new release of the CakePHP Offline Manual. Personally I was extremely happy to download this new release because it finally contains documentation and howtos related to CakePHP’s Model Associations, which is perhaps one of the most used advanced CakePHP features.
So I’ll have no excuses not to learn how to use them, great…

The manual is available in the following formats:

  • Windows compressed HTML (.chm)
  • PDF
  • HTML (multiple pages or single page)

Furthermore, the CakePHP API is also available for download in .chm format, thanks to Mladen Mihajlovic.

Well done guys. Really useful for people like me who are on dialup sometimes :)
