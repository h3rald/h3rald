The H3RALD.com domain was bought in March 2004 and since than has always been my home on the Internet. Through the years, a total of 8 versions of this web site has been released, each featuring some change in the backend.

Currently, this web site is fully static (minus a few Ajax calls) and is powered by the Nanoc, a very flexible web publishing system written in Ruby.
